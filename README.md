# 🚀 CareerOS AI

CareerOS AI is an AI-Powered Placement Readiness Platform designed to help students improve their placement preparation through resume analysis, interview practice, and career tracking.

## 🌟 Features

- 🏠 Home Dashboard
- 📄 Resume Upload
- 📊 Resume Analyzer
- 🎯 Resume Score Tracking
- 💼 Internship Tracker
- 🎤 Mock Interview Preparation
- 📈 Placement Readiness Dashboard

## 🛠️ Technologies Used

### Frontend
- React.js
- JavaScript (ES6)
- HTML5
- CSS3

### Routing
- React Router DOM

### Version Control
- Git
- GitHub

### Deployment
- GitHub Pages
